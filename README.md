## Open Source &amp; Developer Tools

I publish practical tools when I encounter repeated friction in my own workflow or client work. They are small, focused, privacy-aware, and built to solve a specific problem well.

| Project | What it does | Stack |
| --- | --- | --- |
| AI CLI Launchers | A family of VS Code extensions that open AI coding CLIs in a side terminal: [Codex](https://github.com/TheStreamCode/codex-cli-launcher), [Copilot](https://github.com/TheStreamCode/github-copilot-cli-launcher), [Kilo](https://github.com/TheStreamCode/vscode-kilo-cli-launcher), [Antigravity](https://github.com/TheStreamCode/antigravity-cli-launcher), [Grok Build](https://github.com/TheStreamCode/grok-build-launcher). | JavaScript, VS Code |
| [Copilot BYOK Switcher](https://github.com/TheStreamCode/copilot-byok-switcher) | Launcher for GitHub Copilot CLI custom model providers with interactive selection. | JavaScript, VS Code |
| [Agentic R&D Skill](https://github.com/TheStreamCode/agentic-rd-skill) | Structured workflow for agent-assisted research, product, business, and technical planning. | JavaScript, Agent Skills |
